### Project Managment System
This project is a simple project management system built using Node.js, Express, and MongoDB. It allows users to create and manage projects, tasks, and team members.

### Features
- Create and manage projects
- Create and manage tasks
- Manage team members
- Assign tasks to team members
- Track task status (To Do, In Progress, Done)

### Technologies Used
- Node.js
- Express
- MongoDB
- Mongoose
- EJS (for templating)
- Tailwind CSS (for styling)
- React (for frontend)

### Conclusion
This project management system is a great way to keep track of your projects and tasks. It provides a simple and intuitive interface for managing your projects and team members. Whether you're working on a small project or a large one, this system can help you stay organized and on track.
